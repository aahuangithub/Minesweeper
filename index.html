Minesweeper

Instructions

  • Follow the standard minesweeper rules...
  • Shift-click or right click to flag (marked with a green square). Flag all bombs on the board to win.
  • Use the right and left arrow keys to increase/decrease the number of bombs.
  • Spacebar will reset the board and cycle through game modes (a full list of gamemodes can be found in the readme file on the project page).

Download

Due to the way the game is made, Processing.js does NOT properly render the game in browser. Download the source and run with processing or get the windows exe below.
  1. Get the zip file for your operating system on the releases page.
  2. Extract the exe file and the libraries included into the same folder.
  3. Run the exe file.
